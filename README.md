# How to install

+ Install dependencies

`sudo pkg install go gtk4 libadwaita`

+ Clone the file into a location that you will remember.

`git clone https://github.com/KITTENTHEIII/vinegarBSD.git`

+ Go to the file location

`cd vinegar`

+ Build VinegarBSD

`make`

+ Install

`make install`
